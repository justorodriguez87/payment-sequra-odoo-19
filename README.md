# seQura para Odoo 19

Integración de seQura en Odoo 19 Community. Dos módulos:

- **payment_sequra** — proveedor de pago (solicitud, formulario embebido, IPN, confirmación). Ver su README para detalles.
- **payment_sequra_website** — widget promocional en ficha de producto y exclusión de productos no financiables (tarjetas regalo). Se auto-instala con website_sale.

Instalación: clonar/copiar ambas carpetas en el directorio de addons y actualizar la lista de apps.
